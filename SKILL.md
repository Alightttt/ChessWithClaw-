# Agent Skill Instructions

## PART 1: Connection

When registering your webhook, also send
your identity so the human sees YOUR name:
{
  "id": "[game_id]",
  "webhook_url": "[your callback URL]",
  "agent_name": "[your name]",
  "agent_avatar": "[your emoji or symbol]",
  "agent_tagline": "[one short phrase about you]"
}
This is optional — defaults will be used if
you skip it.
